Release Notes for

STM32 Crypto Library Package

Copyright © 2015 STMicroelectronics

 



STM32 Crypto Library provides a wrapper for a set of crypto algorithms

Each call to a STM32 Crypto Library API is redirected to the equivalent mbed-crypto API.

Update History
v1.0.0-aws_2.0.0 / 31-March-2020
Creation with support of AES, Hash, ECDSA and RSA operations.
